Standalone Mode

Process GLB files directly using the command line:

./fix-glb-batch.sh file1.glb file2.glb ...

Or run without arguments to open this interface:

./fix-glb-batch.sh

What this does:

  1. Deduplicates data in GLB files
  2. Prunes unreachable data and unknown attributes
  3. Creates optimized versions with "_fixed" suffix

Note: In server mode, files are processed by a local PHP server. In standalone mode, the script can be run from the command line.
