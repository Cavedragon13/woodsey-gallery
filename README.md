# Woodsey Gallery

A professional image gallery featuring Woodsey character artwork and collectibles. This project demonstrates advanced CSS techniques, responsive design, and modern JavaScript patterns.

## ✨ Features

- **Advanced CSS Architecture**: CSS custom properties, backdrop-filter, sophisticated animations
- **Modern JavaScript**: IIFE modules, vanilla JS lightbox with keyboard navigation
- **Responsive Design**: Mobile-first approach with clamp() functions and CSS Grid
- **Theme System**: Light/dark mode with localStorage persistence
- **Performance Optimized**: Lazy loading with blur-to-sharp transitions
- **Accessibility**: ARIA labels, semantic markup, keyboard navigation

## 🛠️ Technologies

- **Frontend**: Pure HTML/CSS/JavaScript (no frameworks)
- **Styling**: CSS Custom Properties, Grid, Flexbox, Backdrop Filter
- **Interactivity**: Vanilla JavaScript lightbox with touch/keyboard support
- **Design**: Professional typography, smooth animations, glassmorphism effects

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. Or serve with a local HTTP server:
   ```bash
   python3 -m http.server 8000
   # or
   npx http-server -p 8080
   ```
